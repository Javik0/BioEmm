# Imágenes Públicas - BioEmm

Esta carpeta es para imágenes que necesitan ser accesibles directamente por URL.

## Uso:

Los archivos aquí se copian directamente al build sin procesamiento.
Accesibles en: `/images/nombre-archivo.png`

## Ejemplos de uso:

- Fondos de página
- Imágenes para meta tags (Open Graph, Twitter)
- Imágenes que se referencian desde CSS

## Notas:

Para logos e imágenes que se importan en componentes React, 
usa mejor `/src/assets/branding/` para que Vite las optimice.
